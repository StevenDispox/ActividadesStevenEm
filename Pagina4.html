Wampserver

Que son?

WampServer es un ambiente de desarrollo de código abierto que permite crear y probar aplicaciones o páginas web.

Para que sirve?

Es una herramienta de software libre que sirve como servidor web para sistemas Windows. Su utilidad principal es permitir el desarrollo y testing de aplicaciones web en un ambiente local antes de publicarlas en un servidor en línea.
