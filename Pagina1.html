Que son los clientes FTP?

Un cliente FTP es un programa informático que permite acceder mediante protocolos FTP a un servidor para cargar o descargar datos de él.

Para que sirven?

Como pasatiempo o recurso didáctico: Es una forma casual de enseñar a quienes están aprendiendo sobre protocolos de internet. Para transferir grandes cantidades de archivos de servidor de modo interno: Al mover archivos de servidor dentro de un sistema cerrado en alguna organización.

Protocolo

El Protocolo de Transferencia de Archivos (FTP) es un estándar de red utilizado para transferir archivos entre un cliente y un servidor en una red informática. Permite a los usuarios cargar o descargar archivos, así como administrar archivos y directorios en el servidor.
